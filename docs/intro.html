Qu'est-ce que ProjectMaker ?

ProjectMaker est un système qui permet de mettre en avant les petits développeurs et leur projet.

Cette documentation vise à inclure toutes les informations dont vous pourriez avoir besoin concernant ProjectMaker.

info

Documentation à jour !

Vous avez trouvez des fautes, il manque des informations ou juste voulez contribuer, rejoingner notre discord et publier votre corréction !

Commencer​

Voici comment ajouter son projet à ProjectMaker

Rejoindre le discord​

Commencez par rejoindre le discord.

  • Mettre en place son projet
  • Trouver un developper
  • Poster une pub
  • Devenir partenaire
SalonExplication
# ✍・presentationsVous pouvez présenter vos projets à tout le monde ici ! Mais avant de pouvoir poster ici, un staff devra vérifier votre projet
# 🤔・explicationsSi vous avez des questions par rapport aux projets indiqués dans # ✍・presentations , veuillez les écrire dans le salon explication
# 🎬・extrait-de-projetVous pourrez montrer vos extraits de projet, pour les plus intéressés d'entre vous
SalonExplication
# 📑・supportVous êtes ici pour aider, se faire aider et améliorer les projets. Il est possible de mention tout les rôles existants dans le salon # 🎏・roles-projectmaker
# 🔍・cherche-equipierVous chercher des développeurs, designer ou autres, vous pourrez demander ici !

Niveau de difficulter​

Nous avons indiquer sur les tutoriels le niveau de difficulter pour les réaliser, voici plus d'information :

  • :
    Réalisable par tout le monde, ne nécessite aucun savoir particulier
  • :
    Avoir quelques notions dans le domaine
  • :
    Nécessite de la pratique et peut-être un peu compliquer pour les débutants
  • :
    Très difficile à réaliser pour un débutant, (aide d'un membre du staff pour les débutants)
  • :
    N'écéssite beaucoup de connaissances dans le sujet ou beaucoup de pratique, (aide d'un membre du staff pour les débutants)
