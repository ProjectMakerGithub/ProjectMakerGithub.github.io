Qu'est-ce que ProjectMaker ?

ProjectMaker est un système qui permet de mettre en avant les petits développeurs et leur projet.

Cette documentation vise à inclure toutes les informations dont vous pourriez avoir besoin concernant ProjectMaker.

danger

Cette documentation est incomplète. Une annonce sera faite quand elle sera à jour. Merci de votre compréhension

- L'équipe du staff

Commencer​

Voici comment ajouter son projet à ProjectMaker

Rejoindre le discord​

Commencez par rejoindre le discord.

Mettre en place son projet​

# ✍・presentations Vous pouvez présenter vos projets à tout le monde ici ! Mais avant de pouvoir poster ici, un staff devra vérifier votre projet

# 🤔・explications Si vous avez des questions par rapport aux projets indiqués dans # ✍・presentations , veuillez les écrire dans le salon explication

# 🎬・extrait-de-projet Vous pourrez montrer vos extraits de projet, pour les plus intéressés d'entre vous

Chercher un développer​

# 📑・support Vous êtes ici pour aider, se faire aider et améliorer les projets. Il est possible de mention tout les rôles existants dans le salon # 🎏・roles-projectmaker

# 🔍・cherche-equipier Vous chercher des développeurs, designer ou autres, vous pourrez demander ici !

Poster une pub​

Il est l'heure de mettre en valeur votre projet ou votre serveur discord

# 💱・echanges-pubs Ici vous pourrez pub votre Discord GRATUITEMENT, DE L'ARGENT GRATUIT

Devenir partenaire​

Pour avoir plus d'info et connaître les modalitées, merci de lire la page partenaire

Remarques : Votre publicité apparaîtra dans # 🤝・partenaire-liste et sur ce site.

Vous êtes instéressé.e ? Rejoindre le programme de partenaire
